# Domain Glossary

| Term | Definition |
|---|---|
| **ADR** | Architecture Decision Record — documents a design decision with context, choice, and consequences |
| **BibTeX** | Reference management format used with LaTeX for bibliography generation |
| **DRF** | Django REST Framework — used for building the REST API |
| **Gemini** | Google's AI model family; PaperWriter uses Gemini 2.0 Flash |
| **IEEE** | Institute of Electrical and Electronics Engineers — defines the paper format standard |
| **LaTeX** | Document preparation system used for academic paper formatting |
| **MiKTeX** | LaTeX distribution for Windows |
| **pdflatex** | LaTeX compiler that produces PDF output |
| **ProseMirror** | Rich text editor framework that TipTap is built on |
| **SPA** | Single Page Application — the frontend architecture |
| **TeX Live** | LaTeX distribution for Linux/macOS |
| **TipTap** | WYSIWYG editor wrapper around ProseMirror |
| **ViewSet** | DRF class that handles CRUD for a model |
| **WYSIWYG** | What You See Is What You Get — rich text editing |
